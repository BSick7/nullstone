Javascript application core library

## Prerequisites

```
$ npm install -g bower
$ npm install -g fayde-unify
```

## Set up

```
$ git clone git@github.com:wsick/nullstone.git
$ cd nullstone
$ npm install
$ gulp reset # cleans bower libs, bower installs, then symlinks to test and stress bootstrappers
```
